Python GUI-tkinter

发表于 2019-01-20   |   更新于 2019-01-20 | 分类于 python

Python提供了开发GUI的多种选择，在所有的GUI方法中，tkinter是最常用的一种，它是Tk GUI toolkit的pyhon标准接口。使用tkinter能够快速的开发GUI应用，并且非常简单。

创建tkinter GUI的基本步骤

  1. 导入tkinter模块
  2. 创建主窗口（也可以称为容器container）
  3. 往主窗口中添加部件widget
  4. 在部件上应用事件触发器event Trigger

python 3.x中导入的是tkinter模块，而python 2.x中导入的是Tkinter模块。本教程使用的是python 3.x。在创建tkinter GUI应用时要记住两个主要的方法。

  1. Tk(screenName=None,baseName=None,className=’Tk’,useTk=1)：
    创建主窗口,可以通过修改className来重新命名主窗口。
    top = tk.Tk()

  2. mainloop():mainloop就是一个无限循环，让GUI应用一直运行，并且等待事件发生然后处理相应的事件，直到窗口的关闭。
    top.mainloop()

tkinter编写GUI的基本代码结构

窗口与其中的部件是父子关系，包含关系。那么如何组织父窗口中的多个子部件呢？tkinter提供了三种组织部件的几何管理器：

  1. pack():将部件组织成块
  2. grid():按照grid（类似表格）的形式组织部件
  3. place():将部件放在指定的位置。

tkinter中包含多个部件，可以根据需要添加这些部件，下面列出了几个比较常用的部件。

  1. Button部件：按钮部件，给Button绑定函数之后，鼠标点击按钮，程序会作出相应的
    反应。传递相应的参数之后，鼠标点击Button时会具有不同的属性。部件的激活背景色，激活前景色，绑定的函数，按钮上的图像，按钮的宽度，高度都可以通过相应的参数去设置。

  2. Canvas:画布，用来绘制图像或者其它复杂的布局，比如图像，文本及部件等等。Canvas同样有许多配置，可以通过相应的参数去设定。比如bd,bg,cursor,highlightcolor,width,height分别设置边框宽度，画布背景色，鼠标显示形状，高光颜色，画布的宽度，高度。

  1. CheckButton:复选框，通过复选框可以选择任意数目的选项。可以通过Title,activebackground,activeforeground,bg,command,font,image等对复选框的标题，激活背景和前景，背景色，命令，字体，上面的图像进行配置。
  1. Entry:输入框，用来输入单行文本。如果要输入多行文本，可以使用Text部件。参数bd,bg,cursor,command,highlightcolor,width,height分别表示边框宽度，背景颜色，鼠标，绑定函数，高光颜色，宽度，高度。
  1. Frame:框架，用来作为容器，内部可以包含其它部件，框架通常用来组织部件。参数包括highlightcolor,bd,bg,cursor,width,height分别表示高光色，边框宽度，背景颜色，鼠标形状，宽度和高度。
  1. Label:标签，用于显示文本或者图像，随时都可以更新图像或文字。参数包括bd,bg,command,font,image,width,height，分别表示边框宽度，背景颜色，绑定的函数，字体，图像，宽度，高度。
  1. Listbox:列表框，提供列表，用户可以从中选择。参数包括bd,bg,command,font,image,width,height，分别表示边框宽度，背景颜色，绑定的函数，字体，图像，宽度，高度。
  1. MenuButton:菜单按钮，菜单是自顶向下的结构。参数包括：activebackground,activeforeground,bg,command,font,image等对复选框的激活背景和前景色，背景色，命令，字体，上面的图像进行配置。
  1. Menu:菜单，用来创建菜单，参数包括title,activebackground,activeforeground,bg,command,font,image分别表示标题，激活背景、前景色，背景颜色，绑定函数，字体，图像。
  1. Message:消息框，用来显示多行文字，类似于Label。参数bd,bg,font,image,width,height
  1. RadioButton:单选按钮，提供多个选项，但只能选择其中的一个。参数包括activebackground,activeforeground,bg, command,font,image,width,height.

  2. Scrollbar:滑动控制

  3. Text:用来编辑并显示多行文本或者图片。参数包括highlightcolor，insertbackground，bg,font,image,width,height等。

  1. TopLevel:直接由窗口管理器控制，不需要父窗口，参数包括bg,bd,cursor,width,height.
